# Beta 功能

> [!WARNING]
> 本部分的功能目前处于 **Beta** 阶段，API 和配置格式可能随时变化。

Beta 功能是尚在实验阶段的新特性，我们会在此板块中提前发布以收集社区反馈。使用这些功能时请注意：

- 配置格式可能在后续版本中发生变化
- 部分功能可能依赖额外的外部服务
- 欢迎通过 [Issue](https://github.com/HumanAIGC-Engineering/OpenAvatarChat/issues) 反馈问题和建议

## 当前 Beta 功能

### [Chat Agent 模式（OpenClaw 集成）](/beta/chat-agent)

使用多轮工具调用 Agent 替代传统 LLM Handler，为数字人提供工具调用、人格记忆、后台任务协作等增强能力。通过与 [OpenClaw](https://github.com/openclaw/openclaw) 集成，实现更复杂的交互场景。
