# LAM Client Rendering Handler

Client rendering derived from [RTC Client](/en/reference/handlers/client/rtc-client). Supports multi-connection with configurable avatar assets.

## Avatar Selection

```yaml
LamClient:
  module: client/ws_lam_client/ws_lam_client_handler
  asset_path: "lam_samples/barbara.zip"
  concurrent_limit: 5
```
