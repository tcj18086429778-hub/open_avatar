# Beta Features

> [!WARNING]
> Features in this section are currently in **Beta**. APIs and configuration formats may change at any time.

Beta features are experimental capabilities released early for community feedback. When using these features:

- Configuration formats may change in future versions
- Some features may depend on additional external services
- Feedback via [Issues](https://github.com/HumanAIGC-Engineering/OpenAvatarChat/issues) is welcome

## Current Beta Features

### [Chat Agent Mode (OpenClaw Integration)](/en/beta/chat-agent)

Replaces the traditional LLM Handler with a multi-turn tool-calling Agent, providing tool calling, persona memory, and background task collaboration. Integrates with [OpenClaw](https://github.com/openclaw/openclaw) for complex interaction scenarios.
