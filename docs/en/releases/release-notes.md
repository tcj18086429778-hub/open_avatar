# Release Notes

## Version 0.6.0 (2026.04)

- Architecture refactored with frontend/backend separation: [OpenAvatarChat-WebUI](https://github.com/HumanAIGC-Engineering/OpenAvatarChat-WebUI)
- All avatars now support manual interrupt and duplex interrupt modes
- Optimized installation and model download with unified dependency management
- Integrated [SoulX-FlashHead](https://github.com/Soul-AILab/SoulX-FlashHead) diffusion-based avatar

## Version 0.5.1 (2025.08.19)

- LiteAvatar multi-session support
- Added Qwen-Omni multimodal model support via Bailian API

## Version 0.5.0 (2025.08.12)

- Frontend/backend separation with [OpenAvatarChat-WebUI](https://github.com/HumanAIGC-Engineering/OpenAvatarChat-WebUI)
- Basic Dify Chatflow support

## Version 0.4.1 (2025.06.12)

- Added [MuseTalk](https://github.com/TMElyralab/MuseTalk) support with customizable avatars
- 50 new LiteAvatar styles

## Version 0.3.0 (2025.04.18)

- Added [LAM](https://github.com/aigc3d/LAM) digital human support
- Added Bailian API TTS handler
- Added Microsoft Edge TTS support
- Migrated to uv package management

## Version 0.2.2 (2025.04.14)

- 100 new LiteAvatar styles
- GPU backend by default

## Version 0.2.1 (2025.04.07)

- Added history logging
- Text input support
- Camera no longer required at startup

## Version 0.1.0 (2025.02.20)

- Initial modular real-time interactive digital human with cloud API support
